# 🎯 本地开发快速开始

## 最简单的方式 🚀

### Windows 用户
双击运行 `quick-start.bat`

### Mac/Linux 用户
在终端运行:
```bash
./quick-start.sh
```

## 标准方式 📋

### 1. 克隆仓库
```bash
git clone https://github.com/Davidsea-z/devportfolio.github.io.git
cd devportfolio.github.io
```

### 2. 安装依赖
```bash
npm install
```

### 3. 启动开发服务器
```bash
npm start
```

### 4. 访问网站
在浏览器打开: **http://localhost:3000/devportfolio.github.io**

## 新功能预览 ✨

你将看到：
- 🌊 **动态海浪背景** - 平滑的缩放和波浪效果
- 💫 **粒子动画** - 保留原有的粒子效果
- 📱 **响应式设计** - 适配所有设备

## 其他命令 🛠️

```bash
# 构建生产版本
npm run build

# 运行测试
npm test

# 部署到 GitHub Pages
npm run deploy
```
